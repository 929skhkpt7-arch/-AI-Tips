One-time setup

Set it up once, then the trick stays simple

The two setup pieces below connect the short browser command to your full Seeking Alpha chart URL. Copy buttons, browser tabs, and screenshots are included.

A

Create a browser search shortcut

This shortcut tells your browser that anything typed after chart should be sent to a special address — https://chart/%s — where %s is replaced by whatever you type (like adx or nvda).

  1. Go to edge://settings/searchEngines
  2. Click Add next to "Address bar search engines"
  3. Fill in the fields below and click Save
  1. Go to chrome://settings/searchEngines
  2. Under "Site search", click Add
  3. Fill in the fields below and click Add
  1. Safari does not support custom search shortcuts the same way as Edge or Chrome
  2. Instead, create a regular bookmark using https://chart/%s as a placeholder, or use the Redirector rule directly on a bookmark URL such as https://chart/adx
  3. For a closer experience, consider a third-party extension like "Custom Search" from the Safari extension gallery
  1. Right-click any search box and choose Add a Keyword for this Search, or go to about:preferences#search
  2. Add a bookmark with the URL https://chart/%s and set its keyword to chart
  3. Use the keyword from the address bar exactly as you would with Edge or Chrome

Install and configure Redirector

The Redirector extension watches for any address that matches https://chart/... and rewrites it into a full Seeking Alpha charting URL — automatically inserting the ticker you typed.

  1. Go to edge://extensions/
  2. Open the Open Microsoft Edge Add-ons store and search for "Redirector", or install directly from the Chrome Web Store
  3. Install the extension, then open its options page
  4. Click Add to create a new redirect rule and fill in the settings below
  1. Go to chrome://extensions/
  2. Open the Chrome Web Store page for Redirector and click Add to Chrome
  3. Install the extension, then open its options page
  4. Click Add to create a new redirect rule and fill in the settings below
  1. Safari's App Extension model does not support Redirector directly
  2. Use an alternative such as the "Redirect Web for Safari" app from the Mac App Store, which supports similar include-pattern and redirect-to rules
  3. Enter the same pattern and redirect values shown below
  1. Go to about:addons
  2. Search Firefox Add-ons for "Redirector" and install it
  3. Open the extension options and click Add
  4. Fill in the same settings shown below

Optional customization

Change your chart setup, step by step

All chart preferences live in one place: the Redirect to field inside your Redirector rule. Choose one recipe below, make the change, save it, and test with chart adx.

1

First, open the setting you will edit

Open your browser's Redirector extension, choose Edit Redirects, then edit the rule named Seeking Alpha Chart. Find the field labeled Redirect to.

A

Change the chart timeframe

Use this when you want every shortcut chart to open at 1 year or 5 years instead of 3 years.

  1. Where: In the Redirect to field, find interval=3Y.
  2. How: Replace it with one of these values:
For a 1-year chart
interval=1Y
For a 5-year chart
interval=5Y
  1. Save: Click Save in Redirector, then type chart adx in the address bar.
ResultYour ADX comparison chart opens using the new 1-year or 5-year timeframe.
B

Change the comparison benchmarks

Use this when you want to compare the ticker with gold and the S&P 500 instead of the Nasdaq 100.

  1. Where: In the Redirect to field, find the part beginning with compare=.
  2. How: Replace the existing comparison value with:
compare=GLD%2CSP500%2C$1

%2C means a comma. $1 automatically becomes the ticker you typed.

  1. Save: Click Save, then type chart adx.
ResultThe chart opens with ADX, GLD, and the S&P 500 on the same chart.
C

Create a second chart shortcut

Use this when you want to keep the original 3-year chart and add a separate 5-year version.

  1. Browser setting: Add another site-search shortcut named chart5 with this URL:
https://chart5/%s
  1. Redirector setting: Duplicate your Seeking Alpha Chart rule and change its include pattern to:
^https://chart5/([A-Za-z0-9.-]+)$
  1. How: In the duplicated rule's Redirect to field, change interval=3Y to interval=5Y.
  2. Save and test: Save the rule, then type:
chart5 adx
Resultchart adx opens your original chart, while chart5 adx opens the 5-year version.
